Iris Flower Classifier

Predict the species of an Iris flower from its measurements using Machine Learning

About This Project

The classic Iris dataset contains measurements of 150 iris flowers from three species — Setosa, Versicolor, and Virginica. The objective was to build a model that can classify the correct species based on four petal and sepal measurements.

  • Exploration & Visualisation — Analysed class distribution and feature patterns using Seaborn histplots, jointplots, and pairplots to understand how species differ in measurements.
  • Problem Found & Fixed — Detected duplicate rows in the raw dataset. Removed them with drop_duplicates() so the model wouldn't overfit on repeated samples.
  • Label Encoding — Mapped string species labels to integers (Versicolor → 0, Virginica → 1, Setosa → 2) for use with scikit-learn classifiers.
  • Model Training — Trained a Gaussian Naïve Bayes classifier inside a scikit-learn Pipeline. Data split 80 / 20 (train / test) with random_state=42.
  • Evaluation & Deployment — Measured accuracy with accuracy_score, serialised the pipeline with pickle, and deployed it via this Flask REST API.
