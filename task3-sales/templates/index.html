📌 About This Project

This project predicts product sales figures based on advertising budgets spent across three media channels — TV, Radio, and Newspaper. Using the classic Advertising dataset (200 records), we explored how each channel correlates with sales and built a regression model to forecast revenue.

The goal was to help businesses make smarter budget allocation decisions by understanding which advertising medium drives the most sales impact.

Python Pandas Seaborn XGBoost Scikit-learn Flask Pickle

🛠️ What We Did

  • 1
    Data Loading & EDA — Loaded advertising.csv with 200 rows. Analyzed distributions of TV, Radio, and Newspaper spends against Sales using scatter plots and a 3D visualization.
  • 2
    Problem Identified — A simple Linear Regression model showed limited accuracy due to non-linear relationships in the data, especially with the TV channel.
  • 3
    Solution — XGBoost — Switched to an XGBoost Regressor inside a Scikit-learn Pipeline. This handled non-linearity perfectly and achieved a significantly better R² score.
  • 4
    Deployment — Serialized the trained pipeline with Pickle and served it via this Flask REST API with an interactive frontend.
